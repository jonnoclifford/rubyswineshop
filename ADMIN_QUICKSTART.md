# Admin Panel Quick Start Guide

## Getting Started

### First Time Setup
1. Go to **yoursite.com/admin**
2. Click **"Sign in with GitHub"**
3. Allow the application to access your GitHub account
4. You'll be redirected to the admin dashboard

### Daily Use
1. Go to **yoursite.com/admin**
2. Click on the section you want to edit
3. Make your changes
4. Click **"Save Changes"**
5. Done! Changes will be live in 1-2 minutes

## Editing Different Sections

### Business Info Tab
Update your:
- Business name and tagline
- Address and location
- Phone, email, and social media
- Opening hours for each day

**Tip**: For opening hours, you can write "Closed" or "By Appointment" for special cases.

### Hero Section Tab
Edit the big banner at the top of your homepage:
- Main headline (keep it short and catchy)
- Subheadline (describe what makes you special)
- Button text
- Hero images

**Tip**: The headline should be under 60 characters for best impact.

### Wine Menu Tab
Manage your wine list:
- **By the Glass**: Add wines available by the glass
- **By the Bottle**: Organize bottles into categories like "Red", "White & Orange", "Sparkling"

**Actions**:
- Click "+ Add Glass Wine" or "+ Add Wine to Category"
- Click the trash icon to remove a wine
- Click "+ Add Category" to create a new category

**Tip**: Keep tasting notes short - one sentence is perfect.

### Events Tab
Promote your events and special happenings:
- Add recurring events (like "Every Sunday")
- Add one-time events (like "March 15, 2026")
- Include time and detailed description
- Check "recurring" for weekly/monthly events

**Tip**: Use natural language for dates: "Every Sunday" or "First Thursday of the month"

### FAQ Tab
Answer common customer questions:
- Add new questions and answers
- Edit existing FAQs
- Remove outdated questions

**Tip**: Write answers as if you're talking to a friend - friendly and clear.

## Common Tasks

### Changing Opening Hours
1. Go to **Business Info** tab
2. Scroll to **Opening Hours**
3. Update the hours for each day
4. Format: "4:00 PM - 9:00 PM" or "Closed"
5. Click **Save Changes**

### Adding a New Wine
1. Go to **Wine Menu** tab
2. Scroll to either "By the Glass" or find the right category in "By the Bottle"
3. Click **+ Add Wine** button
4. Fill in:
   - Wine name (required)
   - Price (required)
   - Producer (optional)
   - Region (optional)
   - Tasting notes (optional)
5. Click **Save Changes**

### Removing a Wine
1. Go to **Wine Menu** tab
2. Find the wine you want to remove
3. Click the **trash icon** next to it
4. Click **Save Changes**

### Creating a New Event
1. Go to **Events** tab
2. Click **+ Add Event**
3. Fill in:
   - Event title (required)
   - Date/Frequency (required) - e.g., "Every Sunday"
   - Time (optional) - e.g., "2:00 PM - 9:00 PM"
   - Description (required)
   - Check "recurring" if it happens regularly
4. Click **Save Changes**

### Updating Contact Information
1. Go to **Business Info** tab
2. Scroll to **Contact Information**
3. Update phone, email, or Instagram
4. Click **Save Changes**

## Important Notes

### Save Your Changes
Always click **"Save Changes"** at the bottom of each form. Your changes won't be saved until you click this button.

### Preview Before Live
- Click **"Preview Site"** in the top right to see your site
- Changes take 1-2 minutes to appear after saving
- Check your changes on mobile too!

### Required Fields
Fields marked with a red asterisk (*) are required. You'll see an error message if you try to save without filling them in.

### Undo Changes
If you make a mistake:
1. Just reload the page before clicking "Save Changes"
2. Or go back and edit the content again

### Last Saved
The admin panel shows "Last saved: [time]" in the top right so you know when your last change was saved.

## Tips for Best Results

### Writing Tasting Notes
Keep it simple and descriptive:
- Good: "Crisp citrus with floral aromatics"
- Good: "Juicy cherry, light body, minimal tannins"
- Too long: Avoid paragraph-length descriptions

### Event Descriptions
Be clear and inviting:
- Tell people what to expect
- Mention if there's a cover charge (or no cover charge)
- Include any special instructions

### FAQ Answers
Write like you're talking to a customer:
- Be friendly and conversational
- Answer the question completely
- Keep it concise but helpful

### Opening Hours
Be consistent with formatting:
- Use "4:00 PM" not "4pm" or "16:00"
- Write "Closed" for days you're not open
- Include the full time range: "4:00 PM - 9:00 PM"

## Troubleshooting

### "Unauthorized" Error
- Your session expired - log out and log back in
- Make sure you're a collaborator on the GitHub repository

### Can't See My Changes
- Wait 1-2 minutes for deployment
- Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)
- Clear your browser cache

### Save Button Greyed Out
- You may be missing a required field
- Look for error messages in red
- Fill in all required fields (marked with *)

### Changes Disappeared
- Make sure you clicked "Save Changes" before leaving the page
- Check if another admin made changes at the same time

## Getting Help

If you run into issues:
1. Check this guide first
2. Try logging out and back in
3. Contact your developer for technical issues
4. Keep the error message if you see one

## Best Practices

1. **Update regularly**: Keep your wine list and events current
2. **Check mobile**: Preview on your phone - many visitors are mobile
3. **Be consistent**: Use the same formatting style throughout
4. **Save often**: Save your work as you go, especially for big updates
5. **Preview first**: Always preview before announcing changes
